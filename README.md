# Album

This is an ios app created to depict mvvm pattern in a simple way.

Features:
   - MVVM arch pattern with reusable components
   - Multiple API calls and parsing
   - Listing all albums and album information

Using:
   - Swift
   - Alamorire
   - SDWebImage
   - CollectionView
   - MVVM architecture
